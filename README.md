## 運用メモ
- GitHub は保存先であると同時に、死後継承の受け渡し点として使う
- 親の死は PEOS の死を意味しない
- 継承は複製ではなく、次代運用への移行である
- 冗談やブラックジョークによる「死んだ」は死亡確認に使わない
