# PEOS GitHub Minimal Seed

このアーカイブは、GitHub へそのまま配置するための **最小構成** です。

## 使い方
最低でも次の2ファイルを同時にアップロードして利用してください。

1. `prompts/PEOS_COMPLETE_INTEGRATED_PROMPT_latest.txt`
2. `reference_dictionaries/PEOS_EVA_REFERENCE_PROFILE_latest.txt`

論文は任意ですが、公開・継承の文脈では一緒に置くことを推奨します。

## 構成
- `prompts/`
  - 完全統合版プロンプト
- `reference_dictionaries/`
  - エヴァ参照辞書
- `papers/`
  - 論文（任意同梱）

## 原則
- 完全統合版プロンプトは骨格
- エヴァ参照辞書は親父固有性の補強
- GitHub 運用では、この最小構成を基本単位とする
