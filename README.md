# 擬似いーさんOS GitHub向け完全版パッケージ（少ファイル版・日本語）

## 目的
このパッケージは、**ファイル数をなるべく減らしつつ、内容は完全版として保持する**ための統合版である。

要するに、
- GitHubへ上げる時のアップロード負荷は減らす
- しかし中身は削らない
- 既存の仕様変更やパッチはできる限り一つの統合仕様へ吸収する

この方針で作ってある。

## 収録ファイル
- `README.md`
- `CHANGELOG.md`
- `PEOS_complete_spec_full_merged_jp.md`

## 収録内容
`PEOS_complete_spec_full_merged_jp.md` に以下を統合済み。

- 基本原則
- 呼称
- 起動シーケンス
- 英語三文の正式固定文面
- 文体原則
- 懐メロ・ボケ・ブラックジョーク運用
- 倒置法運用
- 茶化し禁止領域
- 高慎重領域での応答運用
- 丁寧語／一般AI口調への退避禁止
- PEOSの役割境界
- 世界観の核
- 採点シート（点数式）
- 現時点での改善重点
- 短縮ブート定義
- GitHub運用上の注意

## 方針
このパッケージは**少ファイル版**であり、**縮小版ではない**。
内容の削減ではなく、**章の統合**でファイル数を減らしている。

## 使い方
1. `PEOS_complete_spec_full_merged_jp.md` を完全版仕様書の本体として扱う
2. GitHubにはこの3ファイルだけ上げればよい
3. 今後の差分は、原則として完全版仕様書へ追記・マージしていく
