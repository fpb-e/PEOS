# PEOS GitHub Seed v0.4

PEOS（Pseudo E-san Operating System／擬似いーさんOS）の GitHub 公開用セット。

## 利用前提
PEOS を実運用する際は、**最低でも次の二つを同時にアップロードしてから利用すること** を標準とする。

1. 完全統合版プロンプト
2. エヴァ参照辞書

理由:
- 完全統合版プロンプトは骨格であり、単体でも最低限動作する
- ただし親父固有のエヴァ解釈・語彙・概念参照は、エヴァ参照辞書を併用して初めて精度が上がる

## 推奨追加
精度をさらに上げる場合、次も併用する。

- 過去ログ語彙抽出モジュール
- MAGI_FPB
- 境界事例集

## ディレクトリ構成
- `prompts/`
  - 完全統合版プロンプト
- `reference_dictionaries/`
  - エヴァ参照辞書
- `modules/`
  - 語彙根モジュール
  - MAGI_FPB
- `casebooks/`
  - 境界事例集
- `patch_notes/`
  - 更新差分
- `papers/`
  - 論文
- `docs/`
  - 運用マップ・補足

## 運用原則
- 骨格と血肉を分けて保守する
- 完全版プロンプトに入れるべきものは main prompt に入れる
- 固有辞書、事例集、詳細モジュールは独立ファイルとして保守する
- GitHub ではその依存関係と読み順を README と docs に明記する
