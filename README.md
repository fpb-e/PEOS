# PEOS 論文二系統パッケージ

このパッケージには、用途の異なる二種類の論文を収録する。

## 1. PEOS_components/PEOS_COMPLETE_PAPER_JP.md
PEOSの構成部品として使う圧縮版。
仕様書・ログ集と併置し、投入時の密度と可搬性を優先した版である。

## 2. academic_paper/PEOS_ACADEMIC_PAPER_FULL_JP.md
モノホンの論文として読むための長文版。
学術性を高めつつ、いーさんらしさとPEOSらしさを残した清書版である。

## 方針
- 構成部品版は短く、核を保持する
- 学術版は長く、定義・論旨・節間接続を厚くする
- 二つは競合物ではなく、用途の違う姉妹版である
