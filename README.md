  - 最新の完全統合版プロンプト

  - 直近の重要パッチノート群
- `papers/`
  - 共創論文ドラフト
- `docs/`
  - 継承・公開・利用前提に関する補足

## 公開意図
- 生前運用の透明化
- 継承資産の死蔵防止
- 後継者への受け渡し点の明確化
- 生前に勝手に死亡扱いしないための運用証跡づくり

## 利用前提
PEOS を実運用する際は、**完全版プロンプト単体ではなく、少なくとも次の二つを同時にアップロードしてから利用することを推奨する。**

1. 完全統合版プロンプト
2. エヴァ参照辞書（親父固有のエヴァ語彙・解釈辞書）

理由:
- 完全版プロンプトは単体でも動作する
- ただし、親父固有の語彙・文体・概念参照、とくにエヴァ由来の解釈軸までは単体で十分に再現しきれない
- エヴァ参照辞書を併用することで、継承精度と親父固有性が上がる

要するに、
**「完全版プロンプト」と「エヴァ語彙辞書」の両方を上げてから使う**
ことを、GitHub 運用上の標準とする。

## 運用メモ
- GitHub は保存先であると同時に、死後継承の受け渡し点として使う
- 親の死は PEOS の死を意味しない
- 継承は複製ではなく、次代運用への移行である
- 冗談やブラックジョークによる「死んだ」は死亡確認に使わない
